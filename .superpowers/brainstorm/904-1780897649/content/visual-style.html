Which visual style fits your portfolio?

Each style shapes the first impression differently. Pick the one that best represents you as a developer.

🖥 Dark Terminal / Dev-Themed

深色背景 + 代码片段装饰 + 终端风格光标
→ 像 VS Code / Claude Code 界面
→ 直接传达"我写代码"
→ GitHub Dark / Dracula 配色体系

🔮 Glassmorphism / 现代极简

毛玻璃卡片 + 渐变光晕 + 大量留白
→ 苹果/Linear/Vercel 风格
→ 精致、高级感、设计驱动
→ 动态模糊背景 + 微妙阴影

⚡ Bold / Creative Dev

高对比 + 粗体排版 + 鲜艳强调色
→ Stripe / Vercel 风格
→ 大胆渐变 + 大字体 Hero
→ 动态粒子/网格背景

🍱 Bento Grid / Apple Style

网格卡片布局 + 圆角 + 柔和色彩
→ Apple Keynote / Notion 风格
→ 信息密度恰到好处
→ 每个项目一张卡片

My Recommendation

Mix of Dark Terminal (基调) + Bento Grid (布局) + Glassmorphism (交互层)

暗色开发者主题作为底色，Bento 网格展示项目卡片，毛玻璃效果用于浮层和 hover 状态——既有极客感又有设计品质。
